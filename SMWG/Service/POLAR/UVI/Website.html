Description

Polar Ultraviolet Imager (UVI) Home Page contains links that describe the instrument, its operation, allow the
user to view 36 second images and key parameter mosaics from the mission and download software, plus links
to the UVI data archive, to lists of UVI publications and presentations, and several miscellaneous mission sites.

InformationURL
Name
Polar UVI Browse Images
URL
http://tideuvira.nsstc.nasa.gov/uvi/brwseimages.html
Description

Browse Image viewer allows the user to scan all the images by specifying a year and day-of-year. The gif plot files contain the
UVI image in geometric coordinates and in magnetic latitude / magnetic local time. At the bottom of each plot is a plot of Polar
Spacecraft altitude in Re versus time for the day with a red line indicating the time/altitude when the image was taken. Below the plot
is list of instrument parameters and spacecraft location. The user can user arrow keys to move back and forth through the images
for a day and use the listed image times (hhmmss.gif) to select a specific image.

InformationURL
Name
UVI KP Image Archive
URL
http://tideuvira.nsstc.nasa.gov/uvi/kpgs-Data.htm
Description

UVI KP Image Archive allows the user to look at all the KP Mosaics for a selected month. The gif or png plot files contain the up to
35 KP images nominally taken with the LBHL filter. Reference images are also provided showing the location of the UVI field of view
relative to continental outlines.

InformationURL
Name
UVI On-line Search
URL
http://tideuvira.nsstc.nasa.gov/uvi/ost/
Description

UVI Online Search Tool (OST) provides an online search and visualization tool of UVI auroral and geophysical metadata covering
the solar cycle 23. Data is available from March 1996 through July 2004. OST is used to find times of UVI image data that match
specific research requirements. Search by time, UVI operational mode, geophysical indices, location, or image content. Search
results can be viewed in tabular form, in summary plots, or as a downloadable text file. Once the search is complete, the digital
image data and software to work with the data can be downloaded.
