Wind EPACT/LEMT 5-min key parameter flux data

ResourceID
spase://VSPO/NumericalData/Wind/EPACT/LEMT/L2/PT5M

Description

This set of key parameter flux data from the Wind EPACT (Energetic Particles: Acceleration,Composition and Transport) telescopes contains fluxes of He, C, O Ne,Si, Fe nuclei in the general range 2-11 MeV/n, each averaged over 5 minutes. Poisson uncertainties are given for each flux.
